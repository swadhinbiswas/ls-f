ls-f

A modern, blazing-fast ls replacement with beautiful file icons, git integration, and tree view.

Get Started View on GitHub

Everything you need from ls

And a lot more that you didn't know you needed.

Native Speed

Written in Rust, matching GNU ls speed even with complex icon rendering and colorization.

Nerd Font Icons

500+ file-type icons using Nerd Fonts for instant visual recognition of any file.

Built-in Tree View

Visualize directory structure with --tree, no extra tools required.

Git Integration

See file status (staged, modified, untracked) inline with --git in any view mode.

GNU Compatible

Supports all standard flags: -l, -a, -R, -h, -t, -S, and 30+ more.

Zero Dependencies

Single static binary (~935 KB). No runtime dependencies, no Python or Bash required.

See it in Action

A modern terminal experience out of the box.

~/my-project
$ lsf
src/   docs/   Cargo.toml   README.md   build.sh
$ lsf --tree --git --depth 2
. [M]
├── Cargo.toml [S]
├── README.md
├── src/ [M]
│ ├── main.rs [M]
│ └── cli.rs
└── docs/
└── INSTALL.md [?]
$ lsf -l --git
drwxr-xr-x 5 user staff 160 Mar 8 10:30 src [M]
-rw-r--r-- 1 user staff 842 Mar 8 09:15 Cargo.toml [S]
-rwxr-xr-x 1 user staff 256 Mar 7 14:22 build.sh

Quick Installation

Choose the method that works best for your environment.

Via Cargo

Recommended — install from crates.io.

cargo install lsf

From Source

Clone and build locally.

git clone https://github.com/swadhinbiswas/ls-f.git cd ls-f cargo install --path .
